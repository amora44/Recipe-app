# My Recipes App

A simple recipe app to different types of recipes.



## Technologies Used

- HTML/CSS
- JavaScript
- MongoDB
- Mongoose
- Express
- Node.js



## Getting Started

A simple app to create recipes. User is able to create new recipe, edit an existing recipe, and delete a specific recipe.
